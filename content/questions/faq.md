### ❓ Why is this project created?

This is a proof-of-concept Hugo site for testing blogs, comparisons, and Q&A sections.

### ❓ What features are included?

- Blog posts with Markdown and code examples
- Comparison tables
- Q&A/FAQ pages
- Minimal Bear Blog theme

### ❓ Can I add more code examples?

Yes! Hugo supports fenced code blocks with syntax highlighting for many languages.
